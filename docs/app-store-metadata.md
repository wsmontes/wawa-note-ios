## Description

Wawa Note is a free, open-source workspace that turns meeting evidence into reusable project memory. Capture audio, scan documents, save links — then let AI extract tasks, decisions, and insights, organized into a typed project graph you can query and explore.

**Your Knowledge. Your Process.™**

There are no Wawa Note servers. You bring your own AI keys — OpenAI, Anthropic, Gemini, DeepSeek, or run a local LLM via LM Studio or Ollama. Export everything. Use your data anywhere. It's yours.

**CAPTURE ANYTHING**
• Record meetings with on-device or Whisper transcription
• Scan documents with VisionKit multi-page OCR
• Write notes, save web bookmarks
• Import Markdown, JSON, PDF, HTML, RTF, ICS, SRT files

**AI-POWERED PIPELINE**
• Automatic extraction → analysis → signal detection → ingestion per item
• AI identifies tasks, decisions, people, risks, and insights
• All results stored locally in your knowledge base

**AGENTIC CHAT**
• Shell-based tool calling — the AI navigates your knowledge like a filesystem
• Context-aware: chat is scoped to projects, items, or globally
• Voice input with on-device speech recognition
• Choice prompts, swipe actions, markdown rendering
• Auto/Deep/Fast agent modes

**PROJECT INTELLIGENCE**
• Task boards with status, priority, and owner tracking
• Typed graph relationships with evidence provenance
• Calendar integration with day summaries
• Project health metrics and signal monitoring

**PROVIDER AGNOSTIC**
Works with any OpenAI-compatible API. Configured providers include OpenAI, Anthropic, Gemini, DeepSeek, plus local models via LM Studio, Ollama, or vLLM.

**OPEN SOURCE**
MIT Licensed. No vendor lock-in. No data collection. You control every aspect of your AI workspace.

## App Review Notes
Wawa Note is an open-source, provider-agnostic AI workspace. The app does NOT collect any user data. Users configure their own AI provider API keys. Audio, transcripts, and analysis all stay on-device. No account required. To test AI features, the reviewer can use Settings → Provider to add any OpenAI-compatible endpoint. On-device features (document scanning, audio recording, note creation) work without any API key.
