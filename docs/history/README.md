# Archived Documentation

These documents were written for the original **Meeting Recorder MVP** (Phases 0-8). They have been superseded by the new direction defined in `docs/deep-research-report.md`.

## Why archived

The project is transforming from a single-purpose meeting recorder into a **local-first AI workspace for project memory** with graph intelligence. These documents reflect the old meeting-centric model and task tracking that no longer represents the current product direction.
